# Anti-Procrastination Sheet (Dr. David Burns)

> **Purpose**: Break the procrastination cycle by making tasks concrete, predicting difficulty/satisfaction, then testing predictions.
>
> **From**: *Feeling Good Handbook* / *Feeling Great* (Daily Mood Log companion tool)
>
> **Instructions**:
>
> 1. List ONE task you've been putting off
> 2. Break it into micro-steps (5-15 min each)
> 3. Predict difficulty (0-100%) and satisfaction (0-100%) for EACH step
> 4. Do ONE step
> 5. Record actual difficulty/satisfaction
> 6. Compare → Learn → Continue

## Cognitive Distortions in Procrastination

*Check which distortions fuel your avoidance:*

- [ ] **All-or-Nothing**: "If I can't do it perfectly, no point starting"
- [ ] **Overgeneralization**: "I never finish things" / "I'm always lazy"
- [ ] **Mental Filter**: Focus only on hard parts; ignore progress
- [ ] **Disqualifying Positive**: "That step was too easy to count"
- [ ] **Jumping to Conclusions**: "It'll take forever" / "I'll fail anyway"
- [ ] **Magnification**: Blowing up difficulty; minimizing ability
- [ ] **Emotional Reasoning**: "I feel overwhelmed → It IS overwhelming"
- [ ] **Should Statements**: "I should already be done" → guilt → avoidance
- [ ] **Labeling**: "I'm a procrastinator" / "I'm lazy"
- [ ] **Personalization**: "My delay proves I'm fundamentally flawed"
