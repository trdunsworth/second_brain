### Tier 2: Yellow Zone (Early Warning - Act Within 24-48 hrs)
*When 2+ warning signs appear*

**Immediate Actions:**
1. **Full Daily Mood Log** for triggering situation(s)
2. **Positive Reframing** on core distress: "What does this struggle show I value?"
3. **Behavioral Activation**: Schedule 2 mastery + 1 pleasure activities TODAY
4. **Reach out**: _______________________________________________ (support person)
5. **Review Toolbox**: Use top 3 rated techniques

### Tier 3: Red Zone (Acute Episode - Intensive Daily)
*When mood drops significantly / functioning impaired*

**Daily Protocol (until 2+ days stable):**
- [ ] **Morning**: Full Mood Log + Pleasure-Predicting for day
- [ ] **Midday**: Quick 3-column check-in
- [ ] **Evening**: Mood Log review + Anti-Procrastination if avoiding
- [ ] **Behavioral Experiment**: Test ONE catastrophic prediction
- [ ] **Contact**: Therapist / trusted person / crisis line if needed
- [ ] **Medication review**: (If applicable) Contact prescriber
